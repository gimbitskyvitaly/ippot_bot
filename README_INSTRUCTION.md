# Инструкция по настройке Telegram бота для записи на тренировки

## Шаг 1: Создание бота в Telegram

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например, "Training Bot")
4. Придумайте username для бота (должен заканчиваться на `bot`, например, `my_training_bot`)
5. Сохраните полученный токен (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Добавление бота в группу

1. Добавьте созданного бота в вашу группу
2. Сделайте бота администратором группы (чтобы он мог отправлять опросы)

### Как узнать ID группы:

**Способ 1: Через бота @userinfobot**
- Добавьте @userinfobot в группу
- Он покажет ID группы (обычно начинается с `-`)

**Способ 2: Через пересылку сообщения**
- Перешлите любое сообщение из группы боту @JsonApiBot или @RawDataBot
- В ответе найдите поле `"chat": {"id": -XXXXXXXXXX}`

**Способ 3: Через веб-версию Telegram**
- Откройте группу в веб-версии Telegram (web.telegram.org)
- Посмотрите на URL: `https://web.telegram.org/k/#-XXXXXXXXXX`
- Число после `#-` и есть ID группы (добавьте минус в начале)

### Как включить ветки (topics) и узнать ID темы:

Если вы хотите, чтобы опросы отправлялись в отдельную ветку "Опросы":

1. Включите режим форума в настройках группы:
   - Настройки группы → Тип группы → Преобразовать в группу с темами
   
2. Создайте тему "Опросы"

3. Чтобы узнать ID темы:
   - Отправьте любое сообщение в эту тему
   - Перешлите это сообщение боту @JsonApiBot
   - Найдите в ответе `"message_thread_id": XXXX`

## Шаг 3: Настройка бота

Откройте файл `training_bot.py` и измените следующие параметры:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"  # Вставьте токен от BotFather
GROUP_ID = "YOUR_GROUP_ID_HERE"    # Вставьте ID группы (с минусом, например: -1001234567890)
THREAD_ID = None                   # Если используете ветки, укажите ID темы, иначе оставьте None
```

Пример:
```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
GROUP_ID = "-1001234567890"
THREAD_ID = 123  # ID темы "Опросы", если используется
```

## Шаг 4: Запуск бота

Запустите бота командой:

```bash
python training_bot.py
```

Бот начнёт работу и будет:
- Отправлять опросы каждый день в 10:00 (проверяя, какой сегодня день)
- **Во вторник** - опрос на пятницу (18:00 БНТУ)
- **В среду** - опрос на воскресенье (10:00 РГУОР)
- Обрабатывать ответы и сохранять их в Excel таблицу

## Шаг 5: Команды бота

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/status` - Показать текущую статистику посещаемости
- `/test_poll` - Отправить тестовый опрос (только для администраторов)

## Как работает бот

1. **Расписание опросов:**
   - Тренировки проходят по пятницам и воскресеньям
   - Опросы отправляются за 4 дня до тренировки
   - Пятница → опрос во вторник
   - Воскресенье → опрос в среду

2. **Формат опросов:**
   - Для пятницы: "Тренировка DD.MM 18:00 БНТУ"
   - Для воскресенья: "Тренировка DD.MM 10:00 РГУОР"
   - Варианты ответов: "иду", "плачу", "не иду"

3. **Excel таблица:**
   - Файл `attendance.xlsx` создаётся автоматически при первом запуске
   - Колонки: "Имя", "Посещаемость (%)", затем даты тренировок
   - "иду" = 1, "плачу"/"не иду" = 0
   - Посещаемость считается автоматически как процент

## Важные примечания

1. Бот должен работать постоянно, чтобы отправлять опросы по расписанию
2. Для работы на сервере используйте:
   ```bash
   nohup python training_bot.py > bot.log 2>&1 &
   ```
   
3. Или используйте systemd, Docker, или другие инструменты для фоновой работы

4. Excel файл будет обновляться автоматически при каждом голосовании
