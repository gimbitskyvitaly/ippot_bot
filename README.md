# Telegram бот для опросов о тренировках

Бот автоматически отправляет опросы в группу за 4 дня до тренировки и ведёт учёт посещаемости в Excel таблице.

## Особенности

- 📅 **Расписание**: Опросы отправляются автоматически (во вторник на пятницу, в среду на воскресенье)
- 📊 **Учёт посещаемости**: Автоматическое ведение Excel таблицы с процентом посещаемости
- 🗳️ **Варианты ответов**: "иду", "плачу", "не иду"
- 💬 **Отправка в ветку**: Поддержка отправки опросов в отдельную тему форума

## Быстрый старт

### 1. Установка зависимостей

```bash
# Установить uv (если ещё не установлен)
curl -LsSf https://astral.sh/uv/install.sh | sh
source $HOME/.local/bin/env

# Клонировать репозиторий и установить зависимости
cd training-bot
uv sync
```

### 2. Настройка бота

1. Получите токен в [@BotFather](https://t.me/BotFather)
2. Добавьте бота в группу как администратора
3. Узнайте ID группы (через @userinfobot или @JsonApiBot)
4. Откройте `training_bot.py` и укажите:
   ```python
   BOT_TOKEN = "ваш_токен"
   GROUP_ID = "-ID_группы"
   THREAD_ID = None  # или ID темы для опросов
   ```

### 3. Запуск

```bash
uv run training-bot
```

Или для фонового запуска:
```bash
nohup uv run python training_bot.py > bot.log 2>&1 &
```
